Real Time Location Tracker

Real-Time Tracker is a web application built with React.js, Node.js,
Express.js, Socket.io, and Leaflet. It allows users to track and share
their real-time locations on an interactive map. The application leverages modern
web technologies to provide a seamless and responsive user experience.

FireBase Integration Form

This project is a web application featuring a contact form integrated
with Firebase for real-time database functionality. The HTML structure
includes a form with fields for the user's name, email, and message,
styled using Font Awesome icons. Upon submission, JavaScript captures the
form data and utilizes Firebase to store this information in the Realtime Database.

Netflix Clone India

This repository contains the source code for a Netflix India
web page created as a practice project. The project aims to
showcase HTML, CSS, and JavaScript skills in building a responsive
and visually appealing website. It replicates a simplified version of
the Netflix India website, allowing users to explore features, read FAQs,
and sign up for membership.
